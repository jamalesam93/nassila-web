The **Manuscript** mode is one connected Ouroboros loop — not seven separate apps.

## Stages

1. **Upload** — DOCX, PDF, or paste text; extract structure and in-text citations.
2. **Cited sources** — Attach PDFs or resolve via Crossref / PubMed / OpenAlex; fetch OA text where available.
3. **Audit** — L1/L2 per reference plus **Sanad** passage grounding when enabled.
4. **Explain** — Mismatch and risk copy inline (Sharh grows here).
5. **Export** — Bibliography and audit summary.

## After an audit

| Panel | Meaning |
|-------|---------|
| **Your passage** | Manuscript text around the cite |
| **Source excerpt** | Abstract or OA chunk sent to Sanad |
| **Claims + quotes** | Verdict per claim with verbatim quotes |
| **Source text** | Abstract only, Open-access full text, or Unavailable |

## Run audit

1. Curate references in Bibliography mode first.
2. Switch to **Manuscript**, upload or paste draft with References section.
3. Enable Sanad in settings if you want L3.
4. **Run audit** and review each cited source.
