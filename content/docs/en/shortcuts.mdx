Nassila is keyboard-friendly. Shortcuts below apply on **Windows** (Ctrl-based).

## Bibliography mode

| Shortcut | Action |
|----------|--------|
| **Ctrl+I** | Import reference files |
| **Ctrl+E** | Export bibliography |
| **Ctrl+Shift+E** | Export CSL JSON |
| **Ctrl+Shift+A** | Run autocorrect |
| **Ctrl+Shift+D** | Find missing DOIs |
| **Ctrl+Shift+V** | Verify references (unified L1+L2) |
| **Ctrl+Shift+U** | Detect duplicates |
| **Ctrl+Z** / **Ctrl+Shift+Z** | Undo / Redo |

## Manuscript mode

Manuscript mode uses the same global shortcuts where applicable (**Import**, **Export**, **Verify** when bibliography drawer is open). Loop-specific actions (**Run audit**, Sanad toggle) are available from the manuscript toolbar and Sanad bar — not all have dedicated shortcuts yet.

<Callout variant="info" title="Menu parity">
If a shortcut does not respond, check the **References** or **File** menu for the same action label.
</Callout>

## Tips

- Run **Verify** only when online; the shortcut still fires but the run may fail or skip without connectivity.
- **Undo/Redo** applies to list edits in Bibliography mode after parsing or manual changes.
- For a full workflow walkthrough, see [How-to guide](/docs/how-to).
