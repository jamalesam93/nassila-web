ناسيلا صديقة للوحة المفاتيح. الاختصارات أدناه تنطبق على **Windows** (مبنية على Ctrl).

## وضع المراجع

| الاختصار | الإجراء |
|----------|--------|
| **Ctrl+I** | استيراد ملفات المراجع |
| **Ctrl+E** | تصدير قائمة المراجع |
| **Ctrl+Shift+E** | تصدير CSL JSON |
| **Ctrl+Shift+A** | تشغيل التصحيح التلقائي |
| **Ctrl+Shift+D** | البحث عن DOI المفقودة |
| **Ctrl+Shift+V** | التحقق من المراجع (L1+L2 موحّد) |
| **Ctrl+Shift+U** | اكتشاف التكرارات |
| **Ctrl+Z** / **Ctrl+Shift+Z** | تراجع / إعادة |

## وضع المخطوطة

يستخدم وضع المخطوطة نفس الاختصارات العامة حيث تنطبق (**Import**، **Export**، **Verify** عند فتح درج المراجع). إجراءات الحلقة (**Run audit**، مفتاح سند) متاحة من شريط أدوات المخطوطة وشريط سند — وليست كلها لها اختصارات مخصصة بعد.

<Callout variant="info" title="تطابق القوائم">
إذا لم يستجب اختصار، راجع قائمة **References** أو **File** لنفس تسمية الإجراء.
</Callout>

## نصائح

- شغّل **Verify** فقط عند الاتصال؛ الاختصار يعمل لكن التشغيل قد يفشل أو يتخطى دون اتصال.
- **Undo/Redo** ينطبق على تعديلات القائمة في وضع المراجع بعد التحليل أو التغييرات اليدوية.
- لجولة عمل كاملة، راجع [دليل الإجراءات](/docs/how-to).
