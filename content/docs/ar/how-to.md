ناسيلا تطبيق سطح مكتب لبناء قائمة المراجع والتحقق منها وفق CSL وتشغيل **تحقق موحّد من السجلات** (L1 + L2) وتصدير قائمة منسّقة.

**التدفق المعتاد:** لصق أو استيراد → حل المعرفات → إصلاح التحقق والتكرار → **تحقق** (متصل) → اختيار نمط CSL → **تصدير**.

## نظرة على النافذة

| المنطقة | الدور |
|---------|--------|
| **الرأس** | تبديل الوضع (مخطوطة / قائمة)، استيراد، تصدير، تحقق |
| **الإدخال** | لصق نص أو BibTeX أو RIS أو CSL-JSON |
| **المخرجات** | صف لكل مرجع مع معاينة منسّقة |
| **المشكلات** | أخطاء التحقق، التكرار، عدم تطابق السجل |

## إضافة المراجع

الصق المراجع أو استورد `.bib` و`.ris` و`.json` و`.docx` و`.pdf` عبر **Ctrl+I**.

استخدم صف المعرفات لـ **DOI** أو **PMID** أو **URL** ثم **حل**.

## التحقق L1 + L2

**Ctrl+Shift+V** عند الاتصال — حتى **200** صف لكل تشغيل.

## التصدير

اختر نمط **CSL** ثم **Ctrl+E**.

## دون اتصال

التحليل والتحرير والتحقق المحلي والتصدير يعملون دون شبكة. **التحقق من السجلات** يتطلب اتصالًا.
