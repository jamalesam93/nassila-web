تسرد هذه الصفحة **عملاً مخططاً مرئياً للمستخدم** وفجوات معروفة. لا تغطي خطوط التدريب أو التقييم الداخلية.

## نضج العمال

| العامل | اليوم | الهدف |
|--------|-------|--------|
| **Raqim** | استيراد Bibliography، L1/L2، تصدير، جسر المراجع | تحسينات citeproc والسجل المستمرة |
| **Sanad** | L3 في حلقة المخطوطة؛ مستويات E4B/12B | تكامل أعمق مع Masdar لمقتطفات النص الكامل |
| **Tasnif / Sharh** | نص حتمي مضمّن | شروح أغنى مع نضج Sharh |
| **Maktab** | Stub | تقسيم تلقائي للمخطوطة في الحلقة |
| **Masdar** | Stub | إرفاق PDF للمستخدم + خط أنابيب مقتطعات المصادر المستشهد بها |
| **Shahid** | Disabled | أدلة الجداول/الأشكال (Tier 3+) |

## حلقة المخطوطة (اتجاه Tier 3)

**اليوم:** رفع/لصق → تحليل References مضمّن → L1/L2 → جلب OA/ملخص → سند عند التفعيل.

**الهدف:** رفع واحد؛ Maktab يقسّم؛ Masdar يستوعب PDF المستشهد بها تلقائياً؛ سند يعمل لكل استشهاد دون لصق يدوي بين الوحدات؛ Shahid للجداول/الأشكال.

<Callout variant="info" title="هياكل صادقة">
يظهر Maktab وMasdar وShahid كفجوات خط أنابيب — وليس تطبيقات منتهية وهمية.
</Callout>

## استشهادات صفحات الويب والويب الرمادي

تحسينات مخططة لمراجع صفحات الويب الفوضوية (وحدات Raqim / Tasnif / Sharh):

### الأساس والموثوقية

- حالة HTTP ونوع المحتوى وتسجيل إعادة التوجيه
- رؤية الروابط الميتة؛ تلميحات جدار الدفع / الأرشيف
- محلّلات خاصة بالمضيف حيث تكون مستقرة (تفضيل الكود على النموذج)
- أعلام ميزات لاقتراحات الذكاء الاصطناعي

### تحسينات التطبيق (لا تحتاج نموذجاً)

- اتساق OpenAlex عندما يبقى `type` هو `webpage`
- نص تاريخ الوصول حسب اللغة
- اقتراحات بيانات وصفية لإعادة الجلب بالدفعات

### بمساعدة النموذج (مستقبلاً)

عند شحن الجوانب المدربة، قد تساعد في:

| السيناريو | ملاحظات |
|----------|-------|
| اقتراح حقول CSL من إشارات الصفحة | المستخدم يؤكد قبل التطبيق |
| تصنيف المنصة / الأدب الرمادي | YouTube ووسائل التواصل وSubstack، إلخ. |
| شرح فشل الجلب | عنوان URL لـ PDF وCloudflare ورابط ميت |

<Callout variant="info" title="سلطة السجل">
يبقى تحقق السجل وتنسيق citeproc في **النواة الحتمية** لـ Raqim — دون استبدال بـ LLM.
</Callout>

## جسر المراجع (مشحون)

v1.1.2+ يرسل References المخطوطة إلى Bibliography ويدعم **التدقيق من مكتبة Bibliography** بمفاتيح استشهاد `manuscript-ref:N`. راجع [جسر المراجع](/docs/bibliography-bridge).

## ما لن نعد به هنا

- استبدال كامل لمدير المراجع
- LLM سحابي كافتراضي (سند المحلي يبقى المسار الأساسي)
- تجاوز تلقائي لجدار الدفع عبر بريد Unpaywall
