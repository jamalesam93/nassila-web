راجع [دليل الاستخدام](/ar/docs/how-to) للخطوات التفصيلية.

## سير العمل

1. إضافة المراجع — لصق أو استيراد في وضع القائمة.
2. حل المعرفات — DOI أو PMID أو URL.
3. إصلاح المشكلات محليًا.
4. تحقق — Ctrl+Shift+V (L1 + L2).
5. تصدير — نمط CSL ثم Ctrl+E.

## Sanad (مطابقة المقاطع)

يتحقق Sanad من الادعاءات مقابل مقتطفات المصدر عبر **نموذج محلي**.

### الإعداد

1. **الإعدادات → مطابقة المقاطع**
2. LM Studio أو Ollama أو vLLM
3. المستوى: **E4B** أو **12B**
4. النماذج: `nassila-sanad-e4b` / `nassila-sanad-12b`

## قبل تدقيق المخطوطة

نظّم المراجع في وضع **القائمة** أولًا، ثم ارجع إلى **المخطوطة** وشغّل التدقيق.

## الشبكة

Crossref وPubMed وOpenAlex وUnpaywall (اختياري). Sanad على جهازك.
